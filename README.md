# POS Kasir App

Aplikasi Point of Sale (POS) untuk kasir menggunakan React + Vite

## Fitur
- Manajemen produk
- Keranjang belanja
- Pencarian & filter produk
- Virtual scrolling untuk performa optimal

## Cara Menjalankan
```bash
# Install dependencies
npm install

# Jalankan development server
npm run dev
```

## Tech Stack
- React
- Vite
- Tailwind CSS (jika pakai)
